# Connefy
Social Network Application

This application has the following functionalities:
- Register;
- LogIn;
- LogOut;
- View Profile;
- Add a friend;
- Remove a friend;
- Chat;
- Add an event;
- Subscribe to an event;
- Unsubscribe to an event;
- See your friends;
- Send a friend request;
- Accept a friend request;
- Decline a friend request;
